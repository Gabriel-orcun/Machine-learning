# Machine Learning From Scratch

Projet personnel visant à implémenter plusieurs algorithmes de machine learning from scratch afin de comprendre profondément les concepts mathématiques et algorithmiques derrière chaque modèle.

## Algorithmes implémentés

- Linear Regression
- Logistic Regression
- Decision Trees
- Neural Networks
- KNN
- SVM (à venir)

## Objectifs

- Comprendre les mathématiques derrière les modèles
- Implémenter les algorithmes sans librairies ML (pas sklearn)
- Visualiser le comportement des modèles
